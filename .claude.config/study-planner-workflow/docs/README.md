## âš ï¸ Troubleshooting

### Claude Code is writing code directly instead of using agents

**Problem:** The workflow is being ignored.

**Solution:**
1. Make sure `.claude/CLAUDE.md` exists
2. Restart Claude Code
3. Try the command again with explicit instruction:
   ```
   /study-planner-workflow <task> (Important: Use Task tool to call agents)
   ```

### Agents are not being called

**Problem:** Task tool is not being invoked.

**Solution:**
- Check that `.claude/CLAUDE.md` contains the workflow instructions
- Make sure you're in a Claude Code session (not regular Claude)
- Try breaking it into steps: "Read /study-planner-workflow and tell me Phase 1"
